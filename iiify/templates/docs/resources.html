Listing Resources

Querying the root of the Internet Archive IIIF v1 (alpha) server results in a list of all publicly available and compatible image and book items. 


  $ curl -i "{{ request.url_root }}"

  HTTP/1.1 200 OK
  Server: nginx/1.4.6 (Ubuntu)
  Date: Tue, 22 Sep 2015 08:07:27 GMT
  Content-Type: application/json
  Content-Length: 21307
  Connection: keep-alive
  Cache-Control: max-age=5

  []

Supported Image Formats

The image formats available for a given item is determine by the item's files on archive.org. To see what formats are available for an item, go to https://archive.org/metadata/:itemid/files. The API is capable of interfacing with jp2 (openjpeg), gif, jpg, tiff, png.
